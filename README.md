A simple laptop built around a Raspberry Pi 5 with 8GB RAM, running Kali Linux. It is overclocked to 2.8 GHz with three selectable overclocking levels for flexible performance tuning. The system is powerful enough for everyday tasks such as coding, office work and web browsing. I added two fans for improved cooling. The housing was designed and 3D-printed by myself — feel free to ask if you need the STL files. The display is an affordable touchscreen from Amazon, so no mouse is required. I use an SSD via USB 3.0 and add an 7735 Display for RAM, CPU, Heat.
